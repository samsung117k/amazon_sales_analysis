# Amazon Sales Analysis

## Overview
In this project, I analyzed Amazon product sales data using Python, SQL, and Tableau to uncover pricing trends,
discount patterns, and customer rating insights across 1,463 products and multiple categories.

## Tools Used
- Python (Pandas) - data cleaning
- SQL - data analysis
- Tableau - data visualization

## Dataset
- Source: [Amazon Sales Dataset - Kaggle](https://www.kaggle.com/datasets/karkavelrajaj/amazon-sales-dataset)
- Size: 1,463 products
- Fields: product name, category, pricing, discounts, ratings, reviews

## Analysis Questions
1. Average discount rate by category
2. Top 10 best rated products
3. Real price vs discounted price by category
4. Discount rate vs ratings relationship
5. Top 10 products with most reviews

## Key Findings
- Computers & Accessories and Electronics categories had the highest average discount rates (80-90%)
- Products with lower discount rates tended to have higher ratings, while heavily discounted products had slightly lower ratings
- Amazon Basics HDMI cables led the most reviewed products, followed by boAt audio products
- Electronics had the highest actual prices on average, along with the largest absolute discounts
